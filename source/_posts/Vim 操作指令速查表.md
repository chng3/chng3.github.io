# 普通模式下基本命令
| 快捷键            | 功能                                       |
| ----------------- | ------------------------------------------ |
| `x`                 | 删除光标处字符                             |
| `dd`                | 删除一个词                                 |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |
| `/search_text`      | 检索文档，在文档后面的部分搜索 search_text |
| `?search_text`      | 检索文档，在文档前面的部分搜索 search_text |
| `n`                 | 移动到后一个检索结果                       |
| `N`                 | 移动到前一个检索结果                       |
|                   |                                            |
| `u`                 | 撤销最后的操作                             |
| `Ctrl + r`          | 重做最后撤销的操作                         |
|                   |                                            |
| `dd`                | 删除当前行                                 |
| `d0`                | 删除到行首                                 |
| `d$`                | 删除到行末                                 |
|                   |                                            |
| `h`               | 光标向左移动一个字符                       |
| `j` 或 `Ctrl + J` | 光标向下移动一行                           |
| `k` 或 `Ctrl + P` | 光标向上移动一行                           |
| `l`               | 光标向右移动一个字符                       |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |

# 进阶之复制和粘贴
| 快捷键            | 功能                                       |
| ----------------- | ------------------------------------------ |
| `yy` | 复制当前行至存储缓冲区 |
| `["x]yy` | 复制当前行至寄存器 x |
| `p`（小写） | 在当前行之后粘贴存储缓冲区中的内容 |
| `P`（大写） | 在当前行之前粘贴存储缓冲区中的内容 |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |
|                   |                                            |
